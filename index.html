Web app accompanying manuscript Hart et al., “Internal climate variability amplifies the need for vector-borne disease outbreak preparedness” PNAS, 2025 (https://doi.org/10.1073/pnas.2507311122). The app is hosted on Hugging Face Spaces. A full-page version of the app is available here. Note that the web app is only intended for exploratory purposes and is not able to handle multiple users simultaneously. If you encounter any issues, please raise an issue on GitHub.
